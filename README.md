ğŸ’¡ Inspiration  
University campuses often face issues with disconnected systems, slow service responses, and repetitive manual queries. Students struggle to find course details, book appointments, or locate staff quickly.  
Our inspiration was to create a **unified AI assistant** that brings together all campus-related services into one intelligent, accessible, and easy-to-use system.

âš™ï¸ What It Does  
SmartCampus is an AI-powered web platform that simplifies campus life by providing:  
- AI Chatbot: Answers natural-language questions about courses, professors, and events.  
- Appointment Scheduler: Helps students book and manage meetings with faculty.  
- Campus Navigation: Provides directions to buildings, labs, and offices.  
- Service Requests: Lets users submit and track administrative or maintenance requests.  
- Feedback System:Collects student input to improve services.  
- Admin Dashboard: Allows staff to manage, monitor, and analyze campus queries.

It acts as a 24/7 virtual helpdesk, reducing the workload on administrative departments while improving accessibility and response time.

ğŸ—ï¸ How We Built It  
The system was built using a modular architecture for scalability and maintainability.  

Technologies Used: 
- Frontend: HTML, CSS, Tailwind (for responsive UI)  
- Backend:FastAPI (Python)  
- Database:MongoDB  
- AI/NLP:LLM + Retrieval-Augmented Generation (RAG) for context-based responses  
- Environment:Docker (for containerized local deployment)  
- Version Control: GitHub  

Each module (Chatbot, Appointment System, and Admin Dashboard) interacts with the backend through secure APIs. The chatbot uses LLM + RAG to fetch relevant answers from the structured campus knowledge base.
